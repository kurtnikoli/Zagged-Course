1. Why Aesthetics Matter

You can have the best hook ever — but if the image is bad, it won't matter.

⚡ The Core Truth

Viral slides = emotion sparking + aesthetically pleasing to watch. You need PERFECT images on every slide — there is no reason not to. Once you master this, it's almost effortless to go viral.

2. The Basics

Three non-negotiable rules for slides that look professional.

1
📐

Image Size

Always use 3:4 ratio. Don't use 9:16 or 1:1 — it looks bad and scrolls weird.

3:4 Only
2
🔤

Fonts

Only use TikTok's "Classic" font. No exceptions. Every other font looks off.

Classic Only
3
📏

Text Placement

Keep text away from faces/products. Use the Rule of Thirds. Text goes top or middle — never bottom.

Rule of Thirds
Do ✅
  • ✓ Put text near the top lines
  • ✓ Leave space around the product
  • ✓ Text above the product, not covering it
  • ✓ Easy to read + keeps focus on product
Don't ❌
  • ✕ Don't put text in the bottom row
  • ✕ Bottom text gets blocked by TikTok UI
  • ✕ Don't cover usernames or buttons
  • ✕ Don't overlap the main subject

4. Keeping Everything Consistent

Your slides should feel like they belong together.

Good Congruency ✅
  • ✓ Text in the same location each slide
  • ✓ Same size and style throughout
  • ✓ Images congruent to the story
Bad Congruency ❌
  • ✕ Video is all over the place
  • ✕ Text in different places / styles
  • ✕ Images not congruent at all

5. Where to Find Good Images

Three methods to source high-quality, organic-looking images.

📌

Pinterest Method

Use "aesthetic" after your keyword. Don't just search "pizza" — search "italy pizza aesthetic".

  • 01 Search keyword + "aesthetic"
  • 02 Pick bright, vibrant, high quality
  • 03 Look for text space
  • 04 SCREENSHOT — don't save
Best Method
🔍

Reddit / Google / IG

Instagram for travel/organic pics. Reddit for niche specific shots like before-and-after.

  • 01 Instagram for lifestyle
  • 02 Google for general
  • 03 Reddit for niche
Good Backup
🤖

AI Method

Upload a reference to ChatGPT, get a Midjourney prompt, then refine until 99% real.

  • 01 Find reference photo
  • 02 ChatGPT → Midjourney prompt
  • 03 Edit prompt details
  • 04 Iterate in Midjourney

6. Improving Bad Images

Low quality photo? Upscale it before you use it.

🔧 Quick Fix

Use free tools like upscale.media to make images look better. Never post a blurry image — there's no excuse.

7. Quick Checklist

Run through this before you post. Every time.

Is every slide 3:4?
Only using TikTok's "Classic" font?
Text not blocking the main subject?
All images high quality and support the story?
Everything consistent across slides?
No random or off-brand pics?
